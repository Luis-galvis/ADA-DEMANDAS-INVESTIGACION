# 🏛️ ADA Litigation Analytics & ML Forecasting

## 📊 Análisis Completo de Demandas ADA con Machine Learning Predictivo

### 🎯 Descripción del Proyecto
Sistema end-to-end para el análisis y predicción de demandas por accesibilidad digital (ADA) en Estados Unidos, incluyendo web scraping, data warehouse, dashboards interactivos y modelos de machine learning.

### 📈 Resultados Principales
- **3,188 casos analizados** del 2024
- **Predicciones hasta 2027** con 85% de precisión
- **Dashboard interactivo** con 15+ visualizaciones
- **4 modelos ML** implementados en BigQuery

### 🚀 Stack Tecnológico
- **Extracción:** Python, BeautifulSoup, Pandas
- **Data Warehouse:** Google BigQuery
- **Visualización:** Looker Studio
- **Machine Learning:** BigQuery ML (ARIMA, Linear Regression)
- **Infraestructura:** Google Cloud Platform

### 📊 Insights Clave Descubiertos
- New York lidera con 50.2% de todos los casos (1,600)
- Fashion & Lifestyle es la industria más demandada (35.2%)
- Víctor Ariza es el demandante más activo (113 casos)
- 22.65% de casos en 2024 involucraron widgets de accesibilidad

### 🔮 Predicciones 2025-2027
- **2025:** 4,187 casos estimados (+31% vs 2024)
- **2026:** 4,600 casos estimados (+10% vs 2025)
- **2027:** 5,060 casos estimados (+10% vs 2026)
- **Widgets:** Alcanzarán 45% de participación en 2026

### 📁 Estructura del Proyecto
📂 Extracción de Datos → 📂 Data Warehouse → 📂 Analytics → 📂 ML → 📂 Dashboards
